\section*{Overview}

Students will write a short scientific paper with the following sections and specifications:
\begin{enumerate}
  \item Abstract
  \begin{itemize}
    \item 1 paragraph ($<$500 words)
  \end{itemize}
  \item Introduction
  \begin{itemize}
    \item $\ge 3$ paragraphs
  \end{itemize}
  \item Methods
  \begin{itemize}
    \item $\ge 3$ paragraphs
    \item Emphasize model description and modeling approach
    \item Include equations
  \end{itemize}
  \item Results
  \begin{itemize}
    \item $\ge 2$ paragraphs
    \item Reference tables and figures
  \end{itemize}
  \item Discussion
  \begin{itemize}
    \item $\ge 2$ paragraphs
  \end{itemize}
  \item Literature Cited
  \begin{itemize}
    \item $\ge 3$ citations
  \end{itemize}
  \item Tables
  \begin{itemize}
    \item $\ge 1$ table
  \end{itemize}
  \item Figures
  \begin{itemize}
    \item $\ge 2$ figures
  \end{itemize}
\end{enumerate}

Ideally, you should analyze your own data.
